# 🧾 Smart Billing System

A desktop-based Smart Billing System developed using Python, Tkinter, and SQLite. This project helps manage products, generate bills, print invoices, and store billing records efficiently.

## 🚀 Features

- 🔐 Login System
- 📦 Product Management
  - Add Product
  - Update Product
  - Delete Product
  - Search Product
- 🛒 Billing System
  - Customer Details
  - Product Selection
  - Quantity Management
  - Automatic Total Calculation
- 🧾 Generate & Print Bills
- 💾 SQLite Database Integration
- 📊 Reports Module

## 🛠️ Technologies Used

- Python 3
- Tkinter (GUI)
- SQLite3
- ReportLab (PDF Bill Generation)

## ▶️ How to Run

1. Clone the repository
2. Install required packages

```bash
pip install reportlab
```

3. Run the project

```bash
python main.py
```
